# PitchPoint Demo Script

## Opening

Roofing contractors need fast roof estimates from an address. PitchPoint is designed to make that workflow feel immediate: address in, roof measured, quote ready.

## Customer Story

1. Start with the address input.
2. Explain that the backend accuracy engine analyzes the property and produces measurement artifacts.
3. Click `Analyze roof`.
4. While loading, point out that the workflow is designed for sales speed.
5. When results appear, show the satellite-style image and animated roof outline.
6. Explain that PitchPoint presents predicted roof area, pitch, squares, and linear-feet line items.
7. Point to the math panel: the calculation is visible so the contractor understands how the number was produced.
8. Point to the estimate bar: this gives a preliminary quote range for materials and labor.
9. Mention that the estimate is preliminary and should be reviewed by a roofing professional.

## Trust Story

PitchPoint does not just show a number. It should show the image, selected structure, assumptions, confidence, warnings, and report output so the contractor can decide whether the result is ready or needs manual review.

## Engineering Story

The accuracy engine produces benchmark and submission artifacts. PitchPoint should make those artifacts visible through an Accuracy Lab:

1. Show the recommended scenario.
2. Show the leaderboard.
3. Show scenario and dataset summaries.
4. Show worst cases and pattern analysis.
5. Explain that the team selected the final approach based on measured error, not guesswork.

## Submission Story

For final submission, PitchPoint should show official test predictions clearly:

1. Address
2. Predicted sqft
3. Confidence
4. Manual review flag
5. Warnings
6. Report link
7. Artifact folder path

## Close

PitchPoint turns roof measurement from a slow report-request process into a polished product flow: enter an address, review the measured roof, and prepare a quote-ready report.

Do not make final accuracy claims until benchmark numbers are available.
